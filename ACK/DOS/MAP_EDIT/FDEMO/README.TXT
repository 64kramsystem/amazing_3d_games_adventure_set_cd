This directory contains everything you need to
view the game map for the FDEMO game using the
DOS version of the map editor.

If you want to change the game map and test it
out with the FDEMO.EXE program, follow these steps:

1. Run map editor by using the FDEMEDIT.BAT program
2. Make any changes to the map you desire
3. Save changes by clicking on the SAVE button in
	the map editor
4. Quit the map editor
5. Run the MKPICS.BAT program to rebuild the game
	resource file PICS.DTF
6. Make sure the new version of the PICS.DTF file
	is in the same directory as the FDEMO.EXE
	program
