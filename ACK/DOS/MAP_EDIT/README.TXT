This directory contains the executable program for
the DOS version of the map editor - MAPEDIT.EXE

Subdirectories are provided with the map data for
the MALL demo and the FDEMO games.

To edit the MALL demo game map, switch to the
MALL subdirectory and type 'MALLEDIT.BAT'

To edit the FDEMO game map, switch to the
FDEMO subdirectory and type 'FDEMEDIT.BAT'

HAVE FUN!
