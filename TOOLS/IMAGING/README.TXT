This directory is here to give you some tools
for creating and manipulating bitmaps for your
own game program.

We have included two different version of
Paint Shop Pro. Version 2 is just the basics,
version 3 has all the bells & whistles.

Try Matisseg for some really nice image
creation tools. You will have to buy the
full version if you want color!

NeoPaint is for you guys who prefer to work
out of DOS. It's one of the best paint
packages there is for MS-DOS!

Now, get to work!
